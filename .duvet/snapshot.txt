    TEXT[!MUST,implication]: then there exists a line L such that:
    TEXT[implication]: - `coverage[L] == Hit`
    TEXT[implication]:   (L is directly reported as executed)
    TEXT[implication]: - L is in the same scope as the annotation's target
    TEXT[implication]: - Every line between L and the annotation's target (exclusive)
    TEXT[implication]:   is classified (`Some`)
    TEXT[implication]:   and has properties that are a subset of
    TEXT[implication]:   {Whitespace, Comment, Annotation, Declaration}
    TEXT[implication]: - No line between L and the annotation's target
    TEXT[implication]:   has the `ScopeClose` property
    TEXT[implication]: - No line between L and the annotation's target
    TEXT[implication]:   has the `ScopeOpen` property
    TEXT[implication]: - No line between L and the annotation's target
    TEXT[implication]:   is unknown (`None`)

    TEXT[!MUST,implementation,implication,test]: annotation B (lines b1..b2)

  SECTION: [Property 7: Target Determinism](#property-7-target-determinism)
    TEXT[implication]: `annotation_target` is a pure function:
    TEXT[implication]: given the same annotation, classifications, and file length,
    TEXT[implication]: it always returns the same result.
    TEXT[implication]: This is free in Verus
    TEXT[implication]: (all `fn` in Verus are deterministic by construction).

  SECTION: [Property 8: Scope Well-formedness](#property-8-scope-well-formedness)
    TEXT[implication]: `build_scope_tree` produces a well-formed scope tree:
    TEXT[implication]: - Every scope has `open_line <= close_line`.
    TEXT[implication]: - If two scopes overlap,
    TEXT[implication]:   one strictly contains the other (proper nesting).
    TEXT[implication]:   No partial overlaps.

  SECTION: [Property 9: Execution Set Containment](#property-9-execution-set-containment)
    TEXT[implication]: If `coverage[line] == Hit`,
    TEXT[implication]: then `line ∈ execution_set(classifications, scopes, coverage)`.

  SECTION: [Property 10: Annotation Target Bounds](#property-10-annotation-target-bounds)
    TEXT[implication]: If `annotation_target(annotation, ...) = Some(target)`,
    TEXT[implication]: then `target.line_number > annotation.end_line`.

  SECTION: [7.4 Properties](#degraded-properties)
    TEXT[!MUST]: These properties MUST be proven with Verus for the degraded path,
    TEXT[!MUST]: alongside the Section 5 properties for the classified path.

  SECTION: [Property D1: Direct Observation](#property-d1-direct-observation)
    TEXT[!MUST,implication]: The implementation MUST prove that the degraded status is a direct observation
    TEXT[!MUST,implication]: of the target line's own coverage, never an inference propagated from another
    TEXT[!MUST,implication]: line.

  SECTION: [Property D2: Degraded Target Bounds](#property-d2-degraded-target-bounds)
    TEXT[!MUST,implication]: The implementation MUST prove that any `Executed` or `NotExecuted` degraded
    TEXT[!MUST,implication]: status resolves a target strictly below the annotation
    TEXT[!MUST,implication]: (`target > annotation.end_line`).

  SECTION: [Property D3: Degraded Stacking Transitivity](#property-d3-degraded-stacking)
    TEXT[!MUST,implication]: The implementation MUST prove that the degraded status depends on the annotation
    TEXT[!MUST,implication]: only through its resolved target: two annotations that resolve to the same
    TEXT[!MUST,implication]: target receive the same status.

  SECTION: [Property D4: Agreement with the Classified Model](#property-d4-agreement-with-classified)
    TEXT[!MUST,implication]: The implementation MUST prove that where the degraded path emits a verdict on a
    TEXT[!MUST,implication]: directly-hit line that a full classifier would mark as a plain `Statement` (and
    TEXT[!MUST,implication]: not `NonLinearControl`), it returns the same result the classified model
    TEXT[!MUST,implication]: `is_annotation_executed` returns, namely `Executed`.
